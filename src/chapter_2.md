# Advanced Usage

## Custom Styling

You can add custom CSS:

```toml
[output.html]
additional-css = ["theme/custom.css"]
```

## Custom JS

```toml
[output.html]
additional-js = ["theme/custom.js"]
```

## Example (Python)

```python
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)
```
